# Task Reminder 插件规格书

## 功能概述

Task Reminder 是一个 Obsidian 插件，提供任务管理和提醒功能，支持从多种数据源聚合今日待办任务。

## 核心功能

### 1. 任务提醒
- **启动时自动提醒**：Obsidian 启动后延迟显示今日待办任务
- **每日只弹一次**：基于 vault 独立的防重复机制
- **提醒样式**：支持通知栏、弹窗或两者都显示
- **手动触发**：可通过命令手动显示今日任务提醒

### 2. 状态栏指示器
- **实时显示**：在状态栏显示待办任务数量
- **点击交互**：点击状态栏可查看详情（仅桌面端）
- **移动端支持**：支持 iOS 和 Android（状态栏功能除外）

### 3. 多数据源支持
- **Daily Note 任务**：从 Daily Note 文件中提取未完成任务
- **Nike 项目任务**：从 Nike 项目文件中提取任务
- **节假日任务**：识别带有 `#holiday` 标签或 `type: holiday` 的任务
- **周期任务**：支持 daily/weekly/monthly 周期任务

### 4. 任务交互
- **会议高亮**：自动识别并高亮 `#meeting` 标签的任务
- **点击跳转**：点击任务直接打开文件并定位到任务行

## 命令规范

### 现有命令

#### Show today's task reminder
- **功能**：手动触发显示今日任务提醒
- **上下文**：全局可用
- **行为**：忽略"已弹过"状态，强制显示

### 待增强命令

#### Move current line task to another date（移动到未来日期）
- **当前行为**：
  - 仅在编辑模式下可用
  - 获取当前光标所在行
  - 验证是否为有效未完成任务（匹配正则 `/^\s*- \[ \]/`）
  - 如果是有效任务，打开日期选择器让用户选择目标日期
  - 将任务从原文件移动到目标日期的 Daily Note 文件

- **增强后行为**：
  - **场景1：当前行是有效任务**
    - 执行现有移动逻辑
    - 从原文件删除任务，添加到目标日期文件
  
  - **场景2：当前行不是有效任务（无选中文本）**
    - 弹出快速添加窗口
    - 用户输入任务内容
    - 选择目标日期
    - 直接添加到目标日期的 Daily Note 文件
  
  - **场景3：有选中文本**
    - 使用选中文本作为任务内容
    - 弹出日期选择器
    - 添加到目标日期的 Daily Note 文件

- **前置条件检查**：
  - 必须配置 Daily Note 路径
  - 当前文件必须在 Daily Note 目录中（仅移动场景需要）
  - 必须有打开的活动文件

- **错误处理**：
  - 未配置 Daily Note 路径：显示"请先在设置中配置 Daily Note 路径"
  - 无打开文件：显示"❌ 没有打开的文件"
  - 非 Daily Note 文件：显示"❌ 仅支持移动 Daily Note 中的任务"
  - 无效任务格式：执行快速添加流程而非报错

## 数据源格式规范

### Daily Note 任务
```markdown
- [ ] 任务内容 #标签
- [x] 已完成任务（不会显示）
```

### Nike 项目任务
```yaml
---
Due Date: YYYY-MM-DD
Done: false
---
```

### 节假日任务
- 包含 `#holiday` 标签
- 或 frontmatter 中 `type: holiday`

### 周期任务配置
```markdown
| 任务名称 | 类型 | 触发条件 | 模式 |
|---------|------|---------|------|
| 任务名 | daily/weekly/monthly | 条件 | replace/accumulate/skip |
```

## 配置选项

### 基础设置
- 启用自动提醒（布尔值，默认 true）
- 延迟时间（秒，默认 30）
- 通知时长（秒，默认 8）
- 提醒样式（枚举：两者都显示/仅通知栏/仅弹窗）
- 显示状态栏（布尔值，默认 true）

### 数据源配置
- Daily Note 路径（字符串）
- Nike 日历路径（字符串）
- 周期任务配置（文件路径）

## 技术约束

- **依赖**：需要 Dataview 插件
- **Obsidian 版本**：v1.4.0 或更高
- **编辑器上下文**：移动任务命令仅在编辑模式下可用
- **文件位置**：移动任务功能仅支持 Daily Note 目录中的文件
